# UIUX-EXPERIMENT6


## AIM

To understand and apply key UX design methods including ideation, user stories, scenarios, flows, and information architecture.
To generate multiple creative low-fidelity ideas using Crazy 8s/brainwriting techniques.
To map user needs through user stories, scenarios, and contextual understanding.
To construct structured flows and site maps that define clear user pathways.

## ALGORITHM

1. Identify a UX problem such as booking a hospital appointment.

2. Conduct a Crazy 8s or brainwriting session to generate eight quick solution sketches.

3. Develop at least five user stories based on personas, goals, and motivations.

4. Select one user story and write a detailed scenario capturing user context and emotions.

5. Choose a task such as ordering food and map its steps using a task flow or user flow.

6. Perform card sorting on a chosen website or app to group information logically.

7. Create a structured site map based on the sorted categories and identified hierarchy.
